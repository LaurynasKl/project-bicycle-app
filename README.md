### Run locally

Would like to run this project locally? Open terminal and follow these steps:

1. Clone this repo

    ```
    https://github.com/LaurynasKl/project-bicycle-app
    ```

2. Instal NPM package
    ```
    npm i
    ```
    or
    ```
    npm install
    ```

3. run the server
    ```
    npm run dev
    ```
